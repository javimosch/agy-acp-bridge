Architecture

ACP Client → agy-acp-bridge → agy (via pty)

The bridge wraps agy's --print mode via a pseudo-terminal, exposing it as an ACP-compliant stdio server. This enables agy to work with any ACP client without modifications.

Usage

Configure your ACP client to launch the bridge: 

{
  "command": "/path/to/agy-acp-bridge",
  "args": ["acp"]
}
acpx universe-agent-acp-client

Installation

cd ~/ai/agy-acp-bridge
./build.sh
